Metric Definitions

Accuracy
(TP + TN) / (TP + TN + FP + FN)
Fraction of all predictions that were correct.
Precision
TP / (TP + FP)
Of every message the model flagged as spam, how many actually were? High precision = fewer false alarms.
Recall (Sensitivity)
TP / (TP + FN)
Of every actual spam message, how many did the model catch? High recall = fewer missed spam.
F1 Score
2 × (Precision × Recall) / (Precision + Recall)
Harmonic mean of precision and recall. Balances both — useful when classes are imbalanced.
